# reset-article-limit
Chrome Extension that resets the monthly free article limit for specific media sites by removing browser data associated with those sites.

# List of Sites (as of v1.0 - Current)
* New York Times
* LA Times
* Washington Post
* Bloomberg
* Wired
* Vanityfair
* Vulture

# Types of Browser Data Removed
* appCache
* Cache
* cacheStorage
* Cookies
* fileSystems
* indexedDB
* localStorage
* pluginData
* serviceWorkers
* webSQL
